## Visão geral

A ZUNO é um conceito de marca criado para transmitir exclusividade, precisão e desempenho. A proposta do site é apresentar a marca de forma elegante, destacando produtos, diferenciais e experiência visual premium.

## Objetivo do projeto

Este projeto foi desenvolvido como um estudo de design e desenvolvimento front-end, com foco em:

- construção de uma landing page autoral
- fortalecimento de identidade visual
- hierarquia visual e composição de layout
- apresentação de produtos de forma premium
- prática com HTML e CSS

## Conceito da marca

A ZUNO não foi pensada como uma marca de custo-benefício, mas como uma marca de alto padrão para músicos que valorizam qualidade, resposta, controle e durabilidade.

Frase-conceito da marca:

**Feito para durar. Projetado para soar.**

## Seções da landing page

### Hero
Apresentação principal da marca com headline de impacto, CTA e imagem destaque.

### Brand Statement
Frase de posicionamento para reforçar a essência da marca.

### Produtos
Grid com categorias principais:
- Pratos
- Bags
- Baquetas

### Por que ZUNO?
Seção com os principais diferenciais da marca:
- Qualidade absurda
- Precisão de fábrica
- Durabilidade
- Sem concessões

### Produto em foco
Destaque visual para o **Ride ZUNO 24”**, apresentado como produto principal da marca.

### Depoimentos
Bloco com feedbacks fictícios de músicos, reforçando percepção de confiança e performance.

### CTA final
Encerramento da página com chamada para explorar os produtos.

## Tecnologias utilizadas

- HTML5
- CSS3
- Google Fonts

## Identidade visual

A identidade da ZUNO foi construída com base em uma estética minimalista, escura e sofisticada.

### Tipografias
- Michroma
- DM Sans
- Cormorant Garamond
